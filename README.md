- The n8n workflow JSON for Chapter 3 (import directly into n8n)

## Why this exists

Kindle does not allow readers to copy text from code blocks. Every snippet in this book
requires exact syntax — one missed backslash in a JSON payload breaks the whole workflow.
This page exists so readers can copy rather than retype.

## Model strings

All snippets use `claude-sonnet-4-6` as written. Anthropic releases new versions regularly.
Verify the current recommended model string at [console.anthropic.com](https://console.anthropic.com)
before deploying any workflow to production.
